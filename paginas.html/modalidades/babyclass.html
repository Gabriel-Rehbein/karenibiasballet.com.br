Baby Class

A Baby Class é a porta de entrada para o mundo da dança. Voltada para crianças pequenas, trabalha coordenação motora, ritmo, musicalidade e socialização de forma lúdica e divertida. As aulas são adaptadas para a faixa etária e incentivam a criatividade, expressão corporal e autoconfiança.
