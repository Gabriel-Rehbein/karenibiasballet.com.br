Ballet

O ballet clássico é uma forma de dança que combina técnica refinada com expressão artística. Trabalha postura, equilíbrio, flexibilidade, leveza e disciplina. Na Karen Ibias Ballet, o ensino é voltado tanto para quem deseja dançar por prazer quanto para quem sonha em seguir carreira profissional. É a base para diversas outras modalidades de dança.
